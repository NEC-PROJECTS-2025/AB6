This project, Chronic Kidney Disease Prediction Using Machine Learning Models, aims to leverage advanced machine learning algorithms to predict the likelihood of chronic kidney disease (CKD) based on patient data. The project involves preprocessing raw medical datasets, applying dimensionality reduction techniques like PCA, and evaluating the performance of multiple models including Decision Tree, Random Forest, XGBoost, and Support Vector Machines.

By analyzing key health indicators such as blood pressure, glucose levels, and creatinine levels, this project seeks to provide a reliable tool for early detection of CKD. The ultimate goal is to support healthcare professionals in decision-making and improve patient outcomes by enabling timely intervention.

This project also includes a user-friendly frontend built using Flask, allowing users to input data and receive predictions seamlessly. This comprehensive approach combines technical precision with practical accessibility to address a critical healthcare challenge.

"This web application is designed to predict chronic kidney disease (CKD) using advanced machine learning models. It provides accurate insights based on user-provided data, helping in early detection and better decision-making. Our goal is to contribute to improved healthcare outcomes through technology."
